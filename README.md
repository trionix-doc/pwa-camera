# 📱 Matter QR Scanner - PWA

Полнофункциональное веб-приложение (PWA) для сканирования QR кодов Matter устройств прямо с телефона.

## ✨ Возможности

- 📷 **Доступ к видеокамере** - работает на всех современных смартфонах и планшетах
- 🔍 **Сканирование QR кодов** - быстрое и точное распознавание QR кодов
- 💾 **Локальное хранилище** - сохранение всех отсканированных кодов в браузере
- 🚀 **PWA** - установка как приложение прямо на домашний экран
- 📱 **Работа offline** - полная функциональность без интернета (кроме внешних CDN)
- 🔦 **Управление вспышкой** - поддержка вспышки камеры
- 🎨 **Темная тема** - автоматическая поддержка темного режима системы
- 📊 **История сканирований** - просмотр, копирование и удаление результатов
- ⚡ **Быстрая работа** - оптимизированное приложение

## 🚀 Быстрый старт

### Установка

1. Откройте приложение в браузере:
   ```
   https://trionix-doc.github.io/pwa-camera/
   ```

2. Нажмите на кнопку меню браузера и выберите "Установить приложение" или "Add to Home Screen"

### Использование

1. **Нажмите кнопку "Начать сканирование"**
   - Браузер запросит разрешение на доступ к камере
   - Разрешите доступ

2. **Наведите камеру на QR код**
   - Зеленый прямоугольник показывает область сканирования
   - При успешном распознании вы услышите звуковой сигнал

3. **Просмотрите результаты**
   - Все отсканированные коды отображаются в списке ниже
   - Нажмите на результат для более подробного просмотра

4. **Управление результатами**
   - 📋 Кнопка "Копировать" - скопировать код в буфер обмена
   - 🗑️ Кнопка "Удалить" - удалить результат из истории

## 🛠️ Технический стек

- **HTML5** - структура приложения
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript (ES6+)** - логика приложения
- **Web APIs**:
  - MediaDevices API для доступа к камере
  - Canvas API для обработки видео
  - localStorage для хранения данных
  - Service Workers для кеширования
  - Web Audio API для звуковых сигналов
- **jsQR** - библиотека для распознавания QR кодов

## 📋 Matter QR Code формат

Приложение поддерживает стандартные QR коды Matter:
- Формат: `MT:...` 
- Пример: `MT:M:1234/S:1234/D:1234`

## 🔒 Приватность

- ✅ Все данные сохраняются **только локально** на вашем устройстве
- ✅ **Нет отправки** данных на серверы
- ✅ Полный контроль над доступом к камере
- ✅ Вы можете отозвать доступ в любой момент в настройках браузера

## 📱 Поддерживаемые устройства

- ✅ iOS 13+ (Safari, Chrome)
- ✅ Android 6+ (Chrome, Firefox, Edge)
- ✅ Desktop (Chrome, Firefox, Safari, Edge)

## ⚙️ Требования

- Современный браузер с поддержкой:
  - MediaDevices API
  - Canvas API
